# Lixstreamingcaio - Lixstream URL Converter

A lightweight cross-platform application built with **Python** and **Flet** to resolve Lixbox (lixstream.com) video IDs or URLs into their actual playable or embeddable media links.

- Convert **Lixbox video ID or URL** into real media links  
- Supports **multiple formats** (m3u8, mp4, or others depending on server response)
- Clean & modern **desktop GUI**
- Copy result directly to clipboard
